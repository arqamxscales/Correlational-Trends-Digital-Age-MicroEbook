# Correlational Trends in the Digital Age

This repository contains the research paper assets and a GitHub Pages site for public viewing.

## Repository contents

- Final draft PDF: `Correlational Trends in the Digital Age Fınal Draft.pdf`
- Editable Pages source: `Copy/Correlational Trends in the Digital Age.pages`
- Landing page: `index.html`

## GitHub Pages

This project is configured to deploy automatically to GitHub Pages using GitHub Actions.

After pushing to `main`, go to:

- **Settings → Pages** and ensure the source is **GitHub Actions**.

Your site URL will be:

`https://<your-username>.github.io/<your-repo-name>/`

## Local preview

Open `index.html` directly in a browser, or use a local static server.
